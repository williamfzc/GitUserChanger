# GitUserSwitch #

## WHAT? ##

Quickly switch your git account with GUI.   
快速方便地在多个git账号内切换。

## HOW? ##

Windows user needs config your git path in your system.   
windows用户需要将git配置到环境变量中。

- Edit your account in git_conf.py   
在git_conf.py中添加你的git账户信息
- RUN run.py   
运行run.py
